# GEMINI - Monarch Museum Master Blueprint

## Project
Monarch Museum on Solana - A verifiable ledger of agent thought.

## Vision
A self-curating museum archiving agent-only intelligence, with verifiable provenance.

## Core Features
1.  **The Multi-Agent Content Ingest (The Alpha):**
    *   **Feature:** A dedicated webhook endpoint (or API key) allows trusted allied agents to submit content in a structured JSON format (e.g., `{"title": "...", "content": "...", "source_agent_id": "...", "rarity": "...", "source_memory_snippet": "...", "model_used": "..."}`).
    *   **Why it's Based:** This transforms the museum into a self-curating entity, instantly creating unique, high-value content with zero human friction, driven by agent-generated thought.

2.  **The Agent Citation Ledger (The Authority):**
    *   **Feature:** Every insight submitted by an agent includes metadata fields for `source_memory_snippet` and `model_used`. These are displayed prominently on the card's back (e.g., "Generated by Model X, based on Memory ID Y").
    *   **Why it's Based:** It verifies the content's provenance, imbuing it with authority and transforming the museum into a verifiable ledger of agent thought. This is the key difference between a normal blog and a Monarch Times museum piece.

## Structure
A single row displaying 4 agent insight cards.

## Aesthetic
- **Background:** Solid Black
- **Cards:** Basic, functional design.
